# Medicine
  - 경구약제 이미지 객체 검출(Object Detection)프로젝트
  - 코드잇 스프린트 AI 엔지니어링 6기 초급 프로젝트용 경진대회
  - kaggle: https://www.kaggle.com/competitions/ai06-level1-project/
  - Overview: 이번 프로젝트의 목표는 사진 속에 있는 최대 4개의 알약의 이름(클래스)과 위치(바운딩 박스)를 검출하는 것
  - Evaluation: mAP@[0.75:0.95] 지표를 사용하여 모델의 성능을 측정
